The Common Data Element (CDE) Curation Tool

Intended for Context administrators, this tool supports the creation and editing of Data Element Concepts, Value Domains, and Data Elements. An account is required to use this tool, and you must have curatorial authority for your Context. This tool's features facilitate the use of the NCICB Enterprise Vocabulary System (EVS) to create administered component names and definitions, helping to ensure ISO/IEC 11179 compliance and also use of caDSR naming conventions.

CDE Curation Tool User's Guide will tell you how to:

  • Search the caDSR for Data Elements, Data Element Concepts, Value Domains, Conceptual Domains, Classification Scheme Items, Value/Meanings, Case Report Form (CRF) Questions, Permissible Values, Object Class and Property

  • Create New, New Using Existing, New Version, Edit and Block Edit Data Elements

  • Create New, New Using Existing, New Version, Edit and Block Edit Data Element Concepts

  • Create New, New Using Existing, New Version, Edit and Block Edit Value Domains

  • Create New Value/Value Meanings
