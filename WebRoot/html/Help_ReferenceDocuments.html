Uploading Attachment Documents to Reference Document
Overview of Reference Document Attachments page
Once you have created a Reference Document for a Data Element, Value Domain or Data Element Concepts, you can attach files to that Reference Document. The attachments are uploaded for Reference Document by using the Upload Document(s) button on the search screen. The Upload Document(s) button brings up the Reference Document Attachments page. On the Reference Document Attachments page you can view any attachments and add or delete attachments to Reference Document in contexts that you have write permissions.

Overal layout of the Upload Reference Document Attachments		  
Viewing Reference Document Attachments
To view an attachment, select the desired link to the right of the Reference Document table. The attached file will open in a new browser window. To save the file locally right click and select “Save Target As…” to save the file to a location on you local computer. Users can view and download all files weather or not they have write permissions to the Reference Documents context.

Uploading Reference Document Attachments
To attach a file to a Reference Document select the desired Reference Document. Only Reference Documents that you have write permissions too will be selectable. Once you have selected the desired Reference Document, use the "Browse…" button to browse the local file. Alternatively, you can type the file name including the path. Click "Upload" to save the file as an attachment to the selected Reference Document. To upload multiple files or one to multiple Reference Documents repeat these steps, as needed. i Please perform a virus scan on files prior to Upload. File scans are not performed by the Curation Tool.
Deleting Reference Document Attachments
To delete an attachment, select the Delete Icon ( r ) next to the desired link. You can only delete attachments of Reference Documents where you have write permissions to that context. A conformation dialog will appear to confirm you action. Select "OK" to delete or "Cancel" to abort the action.
